# Wishes

Single-page anniversary website built with React, HTML, and CSS.

## Run locally

```bash
npm install
npm run dev
```

## Publish to GitHub Pages

1. Push this folder to the GitHub repository `Wishes`.
3. Run:

```bash
npm install
npm run deploy
```

4. In GitHub, open the repository settings and make sure GitHub Pages is set to serve from the `gh-pages` branch if needed.

The Vite base path is already configured for `/Wishes/`.
